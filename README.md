Ce projet consiste à créer une base de données MySQL appelée CinemaDB pour gérer les acteurs, les films, et les relations indiquant quel acteur a joué dans quel film.
Il est compatible avec MySQL et peut être manipulé via Navicat ou tout autre client MySQL.

🛠 Prérequis
MySQL Server
 installé sur votre machine
Navicat for MySQL
 pour gérer la base de données
Python 3.x (si vous utilisez des scripts Python pour interroger la base)
Module Python mysql-connector-python pour la connexion à MySQL
